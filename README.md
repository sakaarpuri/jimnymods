# that SUV Mod

Ideas and resources for modifying your five-door Jimny. Mobile-first visualizer with a 360° viewer, photo-based finish/decal/grille choices, part cards, live ₹ price estimates, build packages, parts links, and a fitting-shop directory for India.

Single static file, no build step, no dependencies.

## Deploy
- **Netlify:** drag this folder onto https://app.netlify.com/drop
- **GitHub Pages:** push repo, enable Pages on main branch
- **Vercel:** `npx vercel` in this folder
